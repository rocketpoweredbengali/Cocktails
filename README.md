# Velvet Pour

Welcome to **Velvet Pour**! This project marks my first venture into frontend web development using the **GSAP (GreenSock Animation Platform)** framework. It has been an incredible journey of learning and creating smooth animations and interactions.

A huge shoutout to **Adrian from JS Mastery** for guiding me through this project and teaching me the fundamentals of GSAP. This project wouldn't have been possible without his tutorials and insights.

---

## 🎯 **Project Overview**
Velvet Pour is a visually engaging and interactive website showcasing my initial exploration of GSAP. The animations are designed to create a fluid, responsive, and delightful user experience.

---

## 🚀 **Features**
- **Dynamic Scroll Animations**: Seamless interactions triggered by scroll events.
- **GSAP-powered Transitions**: Smooth transitions and element animations.
- **Responsive Design**: Optimized for all devices, from desktops to mobiles.
- **Clean and Minimalistic UI**: Aesthetic design focusing on simplicity.

---

## 🛠️ **Technologies Used**
- **GSAP (GreenSock Animation Platform)**: For animations and transitions.
- **HTML & CSS**: To structure and style the website.
- **JavaScript**: For logic and GSAP integration.

---

## 📚 **Learning Experience**
This project has been an incredible learning experience. Key takeaways:
- Understanding the GSAP framework and its potential for frontend animations.
- Implementing scroll-triggered animations for a more interactive user experience.
- Exploring responsive design principles to ensure compatibility across devices.
